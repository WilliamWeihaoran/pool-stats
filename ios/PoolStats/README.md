# PoolStats iOS (SwiftUI + CloudKit)

This is the native iOS port scaffolding. It mirrors the web app data model and uses CloudKit as the source of truth.

## Open in Xcode
1. Open `ios/PoolStats/PoolStats.xcodeproj` in Xcode.
2. Select the `PoolStats` target (blue app icon).
3. Go to **Signing & Capabilities** and add **iCloud**.
4. Check **CloudKit** and choose the default container.
5. Set your team + bundle ID if needed.

Build and run on a simulator or device.
